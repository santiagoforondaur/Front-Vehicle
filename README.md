# Front-Vehicle

# Resumen

Proyecto front-end para la gestión o visualización de vehiculos 
Incluye animaciones implementadas con Anime.js, utilizadas principalmente en la interfaz donde se ingresan los datos de modelo, marca y año

# Tecnologías principales

- Vite → herramienta de desarrollo rápida para proyectos JavaScript/React
- Anime.js → librería para animaciones fluidas

# Estructura del proyecto 

- package.json
- vite
 _ src
 _ components
 _ App.jsx
 _ main.jsx
 _ styles
 public

# Instalacion de dependencias 

- npm install
- npm install anime js

# Activar el codigo 

- npm run dev
